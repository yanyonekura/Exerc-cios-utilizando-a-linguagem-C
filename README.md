# Algoritmos e Estruturas de Dados em C 🚀

Este repositório contém implementações práticas de algoritmos de ordenação e estruturas de dados avançadas, desenvolvidas como parte dos meus estudos de Ciência da Computação na UNIFAP.

O foco aqui é eficiência, análise de complexidade e manipulação de memória em baixo nível utilizando a linguagem C.

## 📊 Algoritmos de Ordenação
Implementações focadas em entender o funcionamento interno e a performance de diferentes métodos:

* **QuickSort:** Divisão e conquista com escolha de pivô.
* **HeapSort:** Utilizando a estrutura de dados Heap para ordenação eficiente.
* **RadixSort & CountingSort:** Algoritmos de ordenação não comparativos (focados em performance com inteiros/CEPs).

## 🌳 Estruturas de Dados Avançadas (Árvores)
Estudo detalhado de estruturas de busca e balanceamento:

* **BST (Binary Search Tree):** Árvore binária de busca padrão.
* **Árvore AVL:** Implementação de rotações para garantir o balanceamento O(\log n).
* **Árvore 2-3:** Estrutura de busca balanceada multidirecional.
* **B-Tree (B-C):** Implementação de nós com múltiplas chaves, fundamental para sistemas de arquivos e bancos de dados.
